## Architecture Summary

**Core Files:**
- `backend/astrology_calc.py` (1070 lines) — Complete Python backend
- `~/.claude/skills/natal-chart/SKILL.md` (384 lines) — Claude Code skill
- `~/.natal-charts/{slug}/chart.json` (~10KB) — Profile data contract

**Dependencies:**
- kerykeion==5.7.2 (pinned)
- python-slugify
- pyswisseph (transitive via kerykeion)
- Python 3.11+

**Key Decisions:**
- Monolithic Python backend (single file for simplicity)
- Non-interactive confirmation patterns (exit codes + --force)
- Offline calculation mode with optional GeoNames online lookup
- Placidus house system (no configurability in v1)
- Traditional planets only for dignities (modern disputed)
